Passaggi per avviare l'applicazione:

Essendo su Eclipse mi basta schiacciare sul tasto verde in alto e selezionare "PrimpProgettoApplication"



Lista degli endpoint disponibili:

/saluto = endpoint di prova per vedere se l'applicazione si è avviata correttamente.



/info = endpoint che restituisce un JSON {"autore": "Giovanni Pacchiarotti"}.



/app-info = endpoint che restituisce nome e versione dell'app e messaggio di benvenuto.



/configurazione-server = endpoint che restituisce il messaggio e la porta del server.
